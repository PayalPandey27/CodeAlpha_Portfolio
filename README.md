# CodeAlpha_Portfolio
A responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases your skills, projects, resume, and contact details in a modern and clean design.

# Features
✔️ Fully responsive design that works on all devices
✔️ Fixed navigation bar for easy section access
✔️ Smooth scrolling between sections
✔️ Scroll-based fade-in animation effect
✔️ Modern and clean UI design
✔️ Separate sections for About, Skills, Projects, Resume, and Contact
✔️ Interactive hover effects on buttons and project cards
✔️ Downloadable resume option
✔️ Simple and lightweight (built with only HTML, CSS, JavaScript)
